Fiber Optic Color Chart Builder

An interactive, browser-based tool for planning and documenting fiber optic splice assignments using the TIA-598-D color code standard. No installation required — everything runs locally in your browser.

The app generates a visual color chart for any standard fiber count (6 to 1728 fibers), lets you map splice "count" labels to specific fiber positions, and exports the finished chart as a PNG image, PDF, printable page, or a fully self-contained HTML file you can share with colleagues.

ℹ️ All data lives entirely in your browser — nothing is uploaded to a server. Use 💾 Save Web Page to preserve your work between sessions.

🚀 Quick Start

Follow these five steps to produce your first completed splice chart in under two minutes.

1
Name your project
Type a project name into the Project Title field in the toolbar (e.g. Node 14 Splice). This label appears in the chart header and is included in exported file names.
2
Set your cable size
Choose the total fiber count from the Cable Size dropdown (6 – 1728 fibers). The chart redraws instantly. Also select the correct Fibers/Ribbon count (12 is standard; 6, 16 MPO, or 24 for specialty cables).
3
Enter count assignments
In the Count Assignments panel on the right, click + Add row, type a label (e.g. A), and fill in a starting fiber number. Add more rows for each count. When ready, press Apply to Chart.
4
Review the chart
Each fiber cell displays the count label on top and the fiber position below. Click any individual fiber cell to manually assign or override a label for just that fiber. Dead fibers appear greyed-out with a dashed outline.
5
Export your chart
Use 💾 Save Web Page to create a shareable HTML snapshot, 📷 PNG for an image, 📄 PDF for a print-ready document, or 🖨️ Print to send directly to a printer.

🖥️ Interface Overview

The app is divided into three main zones: the Toolbar at the top, the Fiber Chart in the main content area, and the Sidebar Panels on the right.

Toolbar

The toolbar sticks to the top of the page as you scroll. It contains all global controls and is split into logical groups separated by thin dividers.

Project Title Text
Free-form text field. The value appears live in the chart header and is embedded in exported PNG/PDF file names and the PDF header line.
Cable Size Select
Total fiber count of the cable: 6, 12, 24, 36, 48, 72, 96, 144, 216, 288, 432, 576, 864, or 1728. Changes take effect immediately and reset the chart.
Fibers / Ribbon Select
Number of individual fibers in each ribbon. Default is 12 (standard loose-tube and ribbon). Use 16 for MPO assemblies, 6 or 24 for specialty ribbons.
🧪 Tube View Toggle
Only visible for cables ≥ 432 fibers. Groups ribbons into six-tube super-bundles (using the TIA Blue–White tube color cycle), making large cables easier to read.
Color Sync Toggle
When ON (default), count assignments are automatically snapped to start on the next TIA-598-D color boundary (multiples of 12). Keeps your chart visually clean.
Color Sep Toggle
Renders a thin colored accent bar between adjacent count assignments so you can visually distinguish groups at a glance without reading every label.
⚠️ Alerts Toggle
Enables strict validation: alerts you if two entries assign the same label and fiber number, or if a count would overflow the cable's total fiber count. Disable to allow loose/exploratory layouts.
🌗 Theme Toggle
Switches between dark (default) and light themes. Your preference is saved in localStorage under the key lumo_fiber_theme and persists across sessions.
Clear All Action
Removes all count assignments and resets the chart to a blank state. The cable size, ribbon size, and toggles are preserved. Cannot be undone.

Fiber Chart

The chart is a color-coded table where each cell represents one fiber. Cells are arranged into rows (ribbons) and groups (tubes or bundles). The column headers show the fiber position range for that column.

Each fiber cell shows three pieces of information:

Location in cell What it shows
Top (orange label) The count label assigned to this fiber (e.g. A, F037). Blank if unassigned.
Middle (large number) The splice count number within the assigned label. Empty dash if unassigned.
Bottom (small number) The absolute fiber position in the cable (e.g. 72).
Bottom color bar When Color Sep is ON, a thin bar shows the label's accent color to group adjacent fibers visually.
💡 Hover over any fiber cell to see it enlarge slightly. Click a cell to open the Assign Count modal for fine-grained individual overrides.

Sidebar Panels

Three stacked panels live in the right-hand sidebar:

Panel Purpose
Count Assignments Where you enter and manage splice count rows. Supports both row-by-row form input and bulk paste mode.
Active Counts A live summary of every label currently plotted on the chart, showing its color swatch, name, and the fiber range it occupies. Click a label name to rename it inline.
TIA-598-D Color Sequence A static legend showing all 12 TIA colors in order (Blue → Aqua). Use this as a quick reference when reading the chart.

📋 Assigning Counts

There are three ways to assign splice counts to fibers. All three produce the same result; choose the method that fits your workflow.

Row Mode (default)

The default input mode in the Count Assignments panel. Each row represents one contiguous block of fibers assigned to a single label.

Column Description
Label Any text string — the splice count identifier (e.g. A, F037, backbone). Use XD, D, or dead to mark dead fibers (see below).
Start # The first count number within that label, or for dead fibers, the absolute fiber position to mark dead.
End # Optional. If filled, creates a range. For regular labels the count increments from Start to End; for dead fibers the position range is marked dead.

Click + Add row to append a new blank row. Click the × button on any row to remove it. Finally, click Apply to Chart to push all rows to the chart.

💡 Click Example to load a sample dataset and see how rows map to the chart. It is a great way to explore the app for the first time.

Paste Mode

Click paste mode in the panel header to switch to a textarea where you can paste a comma-separated list of assignments — ideal when you already have count data in a spreadsheet or text file.

Each line must follow one of these formats:

# Single count — label gets one fiber
LABEL,N

# Range of counts — label gets fibers N through M
LABEL,N-M

# Dead fiber(s) — absolute position(s) to mark as dead
XD,1-6
D,49

A realistic paste example for a 144-fiber cable:

XD,1-6
A,1-12
B,13-24
XD,37-48
C,25-36

After pasting, click Apply. The app parses each line, builds the assignment map, and re-renders the chart. Invalid lines are silently skipped.

⚠️ Label names are case-sensitive: A and a are treated as different splice counts. Dead-fiber keywords (XD, D, dead) are case-insensitive.

Click to Assign (Individual Fiber Override)

Click any fiber cell in the chart to open the Assign Count modal. This lets you assign or override the label for a single fiber, or start a range from that fiber.

Modal field Description
Label The count name to assign to this fiber (and optionally the range).
End fiber Optional. If provided, all fibers from the clicked fiber through this number are assigned the same label.

Press Assign (or hit Enter) to confirm. Press Cancel or Escape to dismiss.

ℹ️ Individual fiber assignments made via the modal are merged into the chart state alongside row-based assignments and are fully preserved in Save Web Page exports.

💀 Dead Fibers

Dead fibers (also called "cuts", "darks", or "non-working fibers") are fibers that are broken, spliced to themselves, or intentionally excluded from use. The app renders dead fibers with a greyscale wash and a dashed outline to make them easy to spot.

To mark fibers as dead, use any of these reserved labels in Row Mode or Paste Mode:

Label value Interpretation
XD Cross-dead (most common). Marks the specified absolute fiber position(s) as dead.
D Dead — shorthand alias for XD.
dead Verbose form — also accepted (case-insensitive).
cut dead Also accepted; shown with the same dead-fiber styling.

Unlike regular count labels, the number(s) provided for dead fibers are interpreted as absolute fiber positions, not splice count numbers. So XD,1-6 marks fibers 1 through 6 as dead at their physical positions in the cable.

⚠️ With Alerts ON, marking the same absolute position as dead more than once triggers a conflict error. Disable Alerts if you need to work around this during exploratory planning.

🎨 TIA-598-D Color Standard

TIA-598-D (ANSI/TIA-598-D) is the North American standard that defines the color coding for optical fiber cables. It assigns a unique color to each fiber position within a 12-fiber ribbon or bundle, then repeats the 12-color cycle for each subsequent group.

The 12 standard colors in order are:

Blue #1
Orange #2
Green #3
Brown #4
Slate #5
White #6
Red #7
Black #8
Yellow #9
Violet #10
Rose #11
Aqua #12

For cables larger than 12 fibers, the color cycle repeats with each new ribbon/tube. For very large cables (432+ fibers), ribbons are further grouped into super-bundles using the same 12-color cycle applied at the tube level — visible in Tube View mode.

ℹ️ The Color Sync toggle enforces alignment to 12-fiber boundaries, keeping your count assignments synchronized with the TIA color groups. This makes charts much easier to read in the field.

⚙️ Toggles Reference

Quick-reference for every toggle in the toolbar:

Toggle Default What it does When to turn off
🧪 Tube View ON (≥432 only) Groups ribbons into 6-ribbon super-bundles with colored tube headers. For cables ≥432 where you prefer a flat list of ribbons.
Color Sync ON Aligns assignment start positions to the nearest 12-fiber boundary (TIA color group). When you have a specific reason for mid-group splits.
Color Sep OFF Adds a thin color bar at the bottom of each fiber cell, color-keyed to its label. When the extra color layer feels visually noisy on dense charts.
⚠️ Alerts ON Validates for duplicate label+position combos and cable capacity overruns before applying. During exploratory planning where rough/overlapping layouts are fine.
🌗 Theme Dark Switches the entire UI to a light color palette. Saved to localStorage. Whenever you prefer dark mode.

🔍 Fiber Position Lookup

The lookup tool sits at the far right of the toolbar. Type any fiber position number into the field and the chart instantly reveals which count label and count number (if any) are assigned to that position.

The result is displayed next to the input field in the format:

Label · count N

If the fiber is unassigned, the result shows a dash (—). If the fiber is dead, it shows the dead status. The matching fiber cell is scrolled into view and highlighted.

💾 Export & Saving

Four export options are available. Each is best suited for a different workflow:

💾
Save Web Page
Downloads a fully self-contained .html file with your entire chart state baked in. Open it in any browser — no internet connection required. This is the recommended way to save and share your work.
📷
PNG Image
Renders the chart area to a high-resolution PNG (2× scale). Best for embedding in emails, tickets, or documentation wikis. File name includes your project title and cable size automatically.
📄
PDF
Creates a landscape A4 PDF with the chart image and a header line containing the project title, standard name, fiber count, and export date. Best for printing or formal deliverables.
🖨️
Print
Opens the browser's native print dialog. The toolbar, sidebar, and modals are automatically hidden. Switch to light theme first for better ink economy.

How Save Web Page works

When you click 💾 Save Web Page, the app serializes the following state into a <script type="application/json"> tag embedded directly in the downloaded HTML:

State key What it stores
cableSize The selected cable fiber count.
fibersPerRibbon The selected ribbon/group size.
projectTitle The project title text field value.
countRowData All rows from the Count Assignments panel.
customColors Any custom label colors set via the color picker.
all toggles The state of all five toolbar toggles.

When you open the saved .html file, the app detects the embedded state tag, parses it, and restores your full session automatically.

💡 You can open a saved .html file, make changes, and re-save it — the new version will contain the updated state.

Custom Label Colors

In the Active Counts panel, click any label name (it has a dashed underline) to open the Change Label Color modal. Pick a custom color to override the default hash-derived color for that label. Click Reset to revert to the default. Custom colors are saved in the web-page export.

🛠️ Troubleshooting

Symptom Likely cause & fix
"COUNT CONFLICT DETECTED" alert Two rows assign the same label + count number. Check for overlapping ranges in the Count Assignments panel, or turn off Alerts.
"ROUTING CAPACITY ERROR" alert The total counts exceed the cable's fiber count. Reduce assignment ranges or increase the Cable Size. Or turn off Alerts to allow overflow.
PNG / PDF export is blank or dark html2canvas may not have finished loading. Wait a moment and try again, or refresh the page if the libraries fail to load from the CDN.
Saved Web Page doesn't restore my data Make sure you opened the downloaded .html file, not the original index.html. The state is embedded in the file at save time.
Color Sync skips my start position This is intentional: Color Sync rounds up to the nearest 12-fiber boundary. Turn off Color Sync if you need exact positioning.
Tube View toggle is not visible Tube View only appears for cables with 432 or more fibers.
Theme resets to dark on reload Theme preference is read from localStorage. If your browser is set to clear site data on close, the preference won't persist. Re-toggle after loading.
Print output is missing colors Enable "Print background graphics" in your browser's print settings. In Chrome: More settings → Background graphics.
